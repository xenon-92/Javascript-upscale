General flow of execution VS Asynchronus way of execution

General flow of execution

console.log(start);
console.log(middle);
console.log(end);



//output
start
middle
end


Asynchronous flow of execution

console.log(start);
setTimeout(function(){ console.log(middle) })
console.log(end);



//output
start
end
middle


Nested callback, disrupting the flow of execution

//pseudocode
doA(
doB();
doC(function(){
doE()
});
doF()
);
doG()


//order of execution
doA() -> doG() --> doB() --> doC() --> doF() --> doE()
